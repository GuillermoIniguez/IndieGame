# IndieGame


Juego Web Básico
Este es un juego web simple donde el jugador debe hacer clic en un cuadro para ganar puntos. El cuadro se mueve a una nueva posición cada vez que se hace clic en él.

Cómo jugar
Abre el archivo index.html en tu navegador.
Haz clic en el cuadro rojo.
Cada clic suma un punto y el cuadro se mueve a una nueva posición.
Tu puntaje se mostrará en pantalla.
